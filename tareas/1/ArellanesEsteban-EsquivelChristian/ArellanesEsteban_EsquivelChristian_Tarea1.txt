El problema que se decidió resolver fue el de "Santa Claus", en donde el lenguaje en el que se desarrolló fue en python a traves de "Visual Studio Code", para ejecutar simplemente es llamarlo desde la terminal ya que el numero de elfos se cambia pero desde el código del programa al igual que el de los renos.
En el código se utilizaron varios métodos de sincronización como lo fueron los mutex en donde lo usamos para proteger el acceso a las variables compartidas por mencionar en el código tenemos a elfos_problema y renos_de_vuelta. También usamos variables de condición que podemos ver en el código mediante santa_cond que esta asociada con el mutex y su función es permitir que los hilos de santa y el de los elfos se sincronicen para que Santa pueda ser despertado bajo las reglas planteadas al inicio del problema, también se hizo uso de semáforos un ejemplo de estos es en elfos_sem que sirve para coordinar la atención que Santa les dará a los elfos con problemas, también aquí tenemos a elfos_sem.acquire que nos sirve para que un elfo que se encuentre en problemas no pueda seguir trabajando hasta que Santa lo ayude.
Adicional a esto se agregaron pequeños detalles al código, por ejemplo aunque el problema no lo menciona una vez Santa se vaya a dejar los regalos el programa terminará ya que el objetivo principal de Santa se ha cumplido, además de esto se colocó un porcentaje en el que los elfos pueden tener un problema (en el código se tiene un 30%) pero este se puede modificar desde el mismo código, además de que colocamos un tiempo en donde se puede observar la hora en que suceden los acontecimientos como si se tratara de una bitácora.
